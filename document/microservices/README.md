# ecommerce-microservices

Scaffold for Tekno e-commerce migration from monolith to microservices.

## Quick start
1. Go to infrastructure folder.
2. Run `docker compose up --build`.
3. Access:
- Gateway: `http://localhost:8080`
- RabbitMQ Management: `http://localhost:15672` (tekno / tekno123)
